Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сайта EvaStyle detailing (далее — Оператор).

2. Собираемые данные

При заполнении формы на сайте мы собираем следующие данные:

  • Имя
  • Номер телефона
  • Информация об автомобиле (по желанию)
  • Выбранная услуга
  • Комментарий (по желанию)

3. Цели обработки

Персональные данные обрабатываются в следующих целях:

  • Связь с клиентом для записи на услугу
  • Информирование о статусе заказа
  • Улучшение качества обслуживания

4. Срок хранения

Персональные данные хранятся в течение 1 года с момента последнего обращения, после чего удаляются. Пользователь может запросить удаление своих данных в любой момент.

5. Передача данных третьим лицам

Оператор не передаёт персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством Российской Федерации.

6. Защита данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от несанкционированного доступа, уничтожения, изменения, блокирования, копирования и распространения.

7. Права субъекта

В соответствии с Федеральным законом №152-ФЗ «О персональных данных» вы имеете право:

  • Запросить информацию о своих персональных данных
  • Потребовать уточнение, блокирование или уничтожение данных
  • Отозвать согласие на обработку персональных данных

Для реализации указанных прав свяжитесь с нами по телефону +7 951 980 87 77.
